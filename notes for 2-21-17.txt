sizeof is size of data type

The sizeof operator gives the size of any data type or
variable:
double amount;
cout << "A double is stored in "
 << sizeof(double) << "bytes\n";
cout << "Variable amount is stored in "
 << sizeof(amount)
 << "bytes\n";
